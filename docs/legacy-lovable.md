## Local development (recommended for production work)

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Requirements: Node.js and npm (for example via [nvm](https://github.com/nvm-sh/nvm)).

## Stack

- Vite, TypeScript, React
- shadcn-ui, Tailwind CSS

## Lovable publish / custom domain

Publishing and custom domains are configured in the Lovable product UI (Share → Publish, Project → Settings → Domains). See [Lovable custom domain docs](https://docs.lovable.dev/features/custom-domain#custom-domain).
